#vulpes-text
- For now, it only works on linux
- It's a text editor
- It's written in go 
- It's a work in progress
- It's not very good
- It's not very fast
- It's not very stable

#how to install
- clone the repo
- run `go build`
- run `./vulpes-text [file]`

#how to use
- use the arrow keys to move the cursor
- use the backspace key to delete characters
- use the enter key to make a new line
- use the escape key to exit

#how to contribute
- make a pull request

#future plans
- add a file explorer
- add a terminal
- add a file manager

#implementation details
- it uses the tcell library for the console interface
- everything is based on that
- it's not very good
